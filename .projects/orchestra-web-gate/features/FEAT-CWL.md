# Cross-tunnel terminal

Web-based terminal (xterm.js) that connects to a tunnel's devtools.terminal tools. Open terminal sessions on any connected machine from the browser. Multiple tabs, one per machine. Uses the tunnel WebSocket for command I/O. Support for streaming output. Session persistence: reconnect to running sessions after page reload.

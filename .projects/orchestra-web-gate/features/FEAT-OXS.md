# Cross-tunnel file operations

Enable file operations across tunnels: browse files on any connected machine, copy files between tunnels (download from source tunnel, upload to target tunnel), compare files across machines. Requires using existing devtools.file-explorer tools. The web app provides a split-pane file browser showing two tunnels side by side. Drag and drop between machines.

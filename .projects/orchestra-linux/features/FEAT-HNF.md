# mTLS certificate configuration

Load mTLS certificates from ~/.orchestra/certs/ directory. CA cert for peer verification, client cert + key for mTLS identity. Use GnuTLS or OpenSSL for TLS operations. Support ed25519 certificates. Fallback to insecure connection for local development when certs are missing. Plugin ID: "ui.linux".

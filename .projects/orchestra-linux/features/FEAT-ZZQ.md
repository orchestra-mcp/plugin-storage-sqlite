# Settings plugin — appearance (theme picker)

SettingsPlugin (id: "settings", icon: "preferences-system-symbolic"). AdwPreferencesWindow with search. Appearance page (AdwPreferencesPage): AdwComboRow for color-theme (25 themes), AdwComboRow for color-scheme (dark/light/system), accent color picker. General page: timezone, language, autostart toggle. AI page: default provider + model AdwComboRow, auto-approve toggle. All settings backed by GSettings schema dev.orchestra.desktop.

# GtkSourceView markdown editor with syntax highlighting

Configure GtkSource.View for markdown editing. Use GtkSource.LanguageManager to load "markdown" language. Create custom "orchestra-dark" GtkSource.StyleScheme matching the Orchestra color palette (syntax_blue, syntax_cyan, syntax_green, syntax_yellow, syntax_orange, syntax_red, syntax_purple). Set monospace font (JetBrains Mono or Fira Code at 13pt). Enable line-wrap (WORD_CHAR), disable line numbers for notes, enable for code files. Content margins: 24px left/right, 16px top/bottom.
